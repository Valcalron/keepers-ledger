# Keeper's Ledger

A personal Graveyard Keeper companion for tracking day-specific NPCs, active tasks, collections, technologies, recipes, alchemy notes, and unlock requirements.

This is intentionally a player checklist and task manager, not a replacement wiki. It is built for the PS5-style use case where the game tells you what to do once and then politely vanishes into the fog.

## Use

Open `index.html` in a browser. Progress is saved locally in the browser with localStorage.

## Features

- Day planner for Moon, Water, Fire, Wind, Tree, and Bone
- NPC availability by day
- Active task board with priority, location, day, and notes
- Museum-style collection checklists for NPCs, technologies, items, recipes, alchemy, and unlocks
- Search and filters
- Export/import save data as JSON
- Seed data that can be edited in the UI
